Que-1:Explain the difference between the following data types with examples: 
• Integer (int)  
• Float (float)  
• String (str)  
• Boolean (bool) 
Answer:-
(a)Integer(int):-Integer is a numeric data type, which are use to store the integer value like (2,23,56).Integer value is denoted by "int".
(b)Float(float):-Float is also a numeric data type which are used to write the decimal number values like (2.5,76.4),Float value is denoted by "float".
(c)string(str):-A string is a fundamental data type used to represent text.it contain text,number and symbols like ("hello","myname123"),string can be written in single,double inverted quotes.
(d)Boolean(bool):-Boolean is a data type which contain value like ("True","False"),boolean work is to print the answer in true or false manner like whether the ans is true or false.

Que-8:Identify whether the following are examples of AI and explain why:
(a)ChatGPT
(b)Google Maps route prediction
(c)Calculator
(d)Netflix recommendations
(e)Voice assistants (Alexa/Siri)
Answer:-
(a)ChatGPT:Yes it is an AI because it understands and generates human-like text.
(b)Google Maps Route Prediction:Yes it is an AI because it analyzes traffic data and predicts the best route.
(c)Calculator:No it is not AI because it only follows fixed mathematical rules and does not learn.
(d)Netflix Recommendations:Yes it is an AI because it suggests movies and shows based on user preferences.
(e)Voice Assistants (Alexa/Siri):Yes they are AI because they understand voice commands and respond to users.
